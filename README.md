## 🛠️ **참여 방법**
```

1. 초대를 받는다.
2. 로컬 저장소로 클론
```
git clone https://github.com/munjh61/SSAFY-CodingTest.git
```
3. 각자 branch를 만들기
```
git checkout -b 브랜치명
```
4. 원격 저장소로 푸시
```
git push origin 브랜치명
```
5. pull request를 꼭 한다.

```
~1. 이 저장소를 fork한다.~ 
~2. 생성된 저장소에서 작업한다.~
